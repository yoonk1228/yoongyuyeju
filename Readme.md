## git clone, pull 이후 실행방법

1. 터미널에 패키지 설치 명령어 입력
```
npm install
```
2. 작업 root영역에 .env 파일 작성 (작업 최상위디렉토리)
.env
```
PORT=
```
3. 서버 정상실행 확인
```
node server
```
node mini project 서버시작, 포트번호 : ${PORT}
